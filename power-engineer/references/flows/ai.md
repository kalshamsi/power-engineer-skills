# AI/LLM Flow

Install AI/LLM engineering and agentic skills. Scans for SDK usage.

## Step 1 -- Scan

Read `references/modules/scanner.md`. Pay attention to SDK detection.

## Step 2 -- Targeted interview

Ask only:
- SDK follow-up: "Which AI SDK are you using?" (if not detected)

Defaults:
- Q1: "AI/LLM Engineering"
- Q2: from scanner
- Q3-Q8: from scanner or "None"

## Step 3 -- Resolve

Read `references/modules/skill-resolver.md`. Include:
- Core methodology
- AI/LLM Engineering project type skills
- SDK-specific skills
- Agentic plugin skills

## Step 4 -- Install

Read `references/modules/installer.md`. Execute installations.

## Step 5 -- Configure

Read `references/modules/configurator.md`. Configure project.
