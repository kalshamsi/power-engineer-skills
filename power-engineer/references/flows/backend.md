# Backend Flow

Install backend, API, and database skills. Scans the project first.

## Step 1 -- Scan

Read `references/modules/scanner.md` and follow its instructions.

## Step 2 -- Targeted interview

Read `references/modules/questionnaire.md` but ONLY ask:

- Q2 Language/stack (if not detected)
- Q7 Cloud/database (if not detected, or to confirm detection)

Defaults for others:
- Q1: "Software Engineering"
- Q4: "None"
- Q5-Q6: "None"
- Q8: from scanner context

## Step 3 -- Resolve

Read `references/modules/skill-resolver.md`. Include:
- Core methodology (always)
- SWE workflow skills
- Backend skills
- Stack-specific skills (from Q2)
- Database skills (from Q7)

## Step 4 -- Install

Read `references/modules/installer.md`. Execute installations.

## Step 5 -- Configure

Read `references/modules/configurator.md`. Configure project.
