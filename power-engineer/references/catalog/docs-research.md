# Documentation & Research

## Document Generation — Anthropic Official

| Skill | Scope | Install | Description |
|-------|-------|---------|-------------|
| docx | `--global` | `npx skills@latest add anthropics/skills/docx --global` | Creates and edits Word documents with headers, tables, tracked changes. |
| pptx | `--global` | `npx skills@latest add anthropics/skills/pptx --global` | Builds polished PowerPoint presentations with layouts, speaker notes, themes. |
| xlsx | `--global` | `npx skills@latest add anthropics/skills/xlsx --global` | Creates and manipulates Excel spreadsheets with formulas, pivot tables. |
| pdf | `--global` | `npx skills@latest add anthropics/skills/pdf --global` | PDF manipulation: text extraction, merging, splitting, form filling, OCR. |
| internal-comms | `--global` | `npx skills@latest add anthropics/skills/internal-comms --global` | Internal communications: status reports, newsletters, FAQs. |
